### 新しい素材ページの仕様

**レイアウト**
- 横スクロールなし（料理レシピページと同じリスト形式）
- 各行に4要素を横並びで表示：
  1. 素材名（日本語 + 英語）
  2. 種別バッジ（魚 / 果物 / 野菜 など）
  3. 使用レシピ件数（タップでレシピページへ遷移）
  4. 解放トグルボタン（✓ / —）

**タップで詳細展開**
- 入手方法
- 在庫（− / 数字 / ＋ ステッパー）

**フィルター機能**
- テキスト検索（素材名で絞り込み）
- 解放状態フィルター：すべて / 解放済み / 未解放
- 種別フィルター：すべて / 魚 / 果物 / 野菜 など
